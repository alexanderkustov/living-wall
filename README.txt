###README###

Olá pessoas! Como podem ver o código está muito mais bonito e vi-me livre de bastante código desnecessário.

Os cubos já estão todos númerados e encontram-se em js/cubes.js
Para registar a venda de um cubo basta acrescentar elementos ao array em js/soldcubes.js

O cubo pode levar uma capa com um logotipo por exemplo, mas devido a um bug no THREE.js é necessário estar a correr num webserver para funcionar.
Basta arrastarem o conteúdo para a pasta do XAMPP e acederem dessa forma.
